# Contexto del proyecto - BC Soluciones Digitales

## Resumen

BC Soluciones Digitales es un proyecto comercial orientado a crear sitios web, automatizaciones, asistentes con IA y sistemas digitales para negocios.

La primera version publica de la web ya esta desplegada en:

https://bcsolucionesdigitales.com

El proyecto usa un servidor propio como infraestructura principal.

## Objetivo actual

El objetivo actual no es solo tener una landing, sino convertir la web en la primera pieza de un sistema comercial mas completo:

- captar potenciales clientes;
- permitir contacto rapido por WhatsApp;
- permitir contacto por formulario;
- permitir llamadas directas;
- guardar leads;
- automatizar respuestas iniciales;
- agendar reuniones;
- generar resumenes comerciales;
- llegar a reuniones con contexto.

## Decision principal de canal

El canal principal de atencion sera WhatsApp.

La web no debe depender de un chatbot incrustado pesado.

La web debe ser una vitrina rapida y clara que derive a:

- WhatsApp del bot;
- formulario de contacto;
- llamada directa a Benjamin;
- llamada directa a Carlos;
- correo.

## Servicios iniciales

Los servicios base de BC Soluciones Digitales son:

1. Landing page profesional
   - Web simple para presentar servicios, contacto, WhatsApp y presencia online.

2. Automatizacion con IA
   - Asistentes y flujos automatizados para responder clientes, ordenar informacion y apoyar procesos internos.

3. Sistema web a medida
   - Aplicaciones internas, paneles, formularios, gestion de clientes, ordenes o procesos especificos.

4. Integraciones digitales
   - Conexion entre formularios, WhatsApp, bases de datos, calendarios, correos y otros sistemas.

## Producto inicial deseado

El primer producto real sera un asistente comercial para ByC.

Debe poder:

- responder preguntas sobre servicios;
- pedir datos basicos;
- detectar intencion comercial;
- guardar leads;
- resumir lo que necesita el potencial cliente;
- proponer reunion;
- revisar disponibilidad en calendario;
- agendar reunion;
- dejar contexto para Benjamin/Carlos.

## Enfoque comercial

No vender "IA generica".

Vender soluciones concretas para negocios:

- mas contactos;
- mejor orden;
- menos tareas repetitivas;
- atencion mas rapida;
- presencia digital profesional;
- automatizacion practica.
