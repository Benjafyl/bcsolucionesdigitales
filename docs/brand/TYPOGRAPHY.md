# Tipografía

## Tipografías recomendadas

### Principal para títulos y marca

`Sora`

Usar `Sora` para:

- títulos principales;
- titulares de secciones;
- elementos de marca;
- composición del logo cuando corresponda;
- llamados visuales importantes.

### Principal para textos e interfaz

`Inter`

Usar `Inter` para:

- párrafos;
- descripciones;
- botones;
- formularios;
- navegación;
- etiquetas;
- componentes de interfaz.

## Alternativas aceptables

Si por disponibilidad técnica no puede usarse la combinación principal, se aceptan:

- `Manrope`
- `Plus Jakarta Sans`
- `Montserrat`, con moderación
- `General Sans`, si está disponible

## Combinación recomendada

```text
Sora + Inter
```

## Jerarquía tipográfica sugerida

### H1

- `Sora`
- peso alto
- uso corto y claro
- máximo protagonismo visual

### H2

- `Sora`
- peso alto
- lectura limpia
- buena separación respecto del cuerpo

### H3 / subtítulos

- `Sora` o `Inter` en peso semibold/bold, según contexto
- suficiente contraste visual sin competir con H1/H2

### Párrafos

- `Inter`
- lectura simple y profesional
- tamaño cómodo
- alta legibilidad

### Botones

- `Inter`
- peso medium o semibold
- texto corto, claro y directo

### Formularios

- `Inter`
- claridad y legibilidad primero

## Reglas generales

- Mantener jerarquía consistente.
- Evitar exceso de pesos tipográficos distintos.
- No usar tipografías decorativas.
- No mezclar demasiadas familias.
- No usar estilos que vuelvan la marca informal.

## Lo que se debe evitar

- tipografías gamer;
- tipografías futuristas exageradas;
- tipografías infantiles;
- títulos demasiado estrechos o poco legibles;
- combinaciones tipográficas con demasiada personalidad visual.

## Criterio de validación

Una buena implementación tipográfica debe sentirse:

- moderna;
- limpia;
- confiable;
- clara;
- profesional;
- fácil de leer.
