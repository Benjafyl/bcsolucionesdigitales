# Paleta de Colores

## Paleta oficial

La paleta oficial de `B&C Soluciones` está basada en tonos sobrios, oscuros y profesionales.

| Nombre | Hex | Uso principal |
| --- | --- | --- |
| Verde oliva profundo | `#5E6F5B` | Color principal de marca, botones, acentos, elementos destacados e isotipo |
| Negro carbón | `#111411` | Fondo principal oscuro, secciones principales y base visual del sitio |
| Gris grafito | `#242924` | Cards, fondos secundarios, bloques de contenido y separadores |
| Blanco hueso | `#F3F4EF` | Texto principal sobre fondo oscuro, fondos claros y versiones limpias del logo |
| Gris piedra | `#A7AEA5` | Texto secundario, descripciones, detalles y bordes sutiles |
| Verde salvia | `#8FA28A` | Acento suave, estados hover, detalles secundarios, íconos o elementos complementarios |

## Roles por color

### Verde oliva profundo

- color principal de marca;
- color del isotipo;
- botones principales;
- acentos visuales;
- detalles que necesiten identidad.

### Negro carbón

- fondo principal;
- navegación superior;
- secciones hero;
- base general del sitio.

### Gris grafito

- cards;
- bloques secundarios;
- paneles;
- fondos de secciones interiores.

### Blanco hueso

- texto principal en fondos oscuros;
- fondos claros;
- superficies limpias en branding secundario;
- variantes claras del logo.

### Gris piedra

- texto secundario;
- descripciones;
- etiquetas;
- bordes sutiles;
- metadatos visuales.

### Verde salvia

- hover;
- estados suaves;
- detalles de apoyo;
- íconos secundarios;
- acentos discretos.

## Uso recomendado en web

| Elemento UI | Color recomendado |
| --- | --- |
| Fondo principal | `#111411` |
| Bloques secundarios / cards | `#242924` |
| Texto principal | `#F3F4EF` |
| Texto secundario | `#A7AEA5` |
| Botones principales | `#5E6F5B` |
| Hover / detalles suaves | `#8FA28A` |

## Reglas de uso

- El fondo oscuro debe ser dominante.
- El verde oliva debe usarse como acento estratégico, no como relleno excesivo.
- El contraste debe ser claro y elegante.
- El verde salvia debe usarse de forma sutil.
- El blanco hueso debe priorizarse por sobre blancos puros agresivos cuando se busque una sensación más sobria.

## Lo que no se debe hacer

- No inventar nuevos colores base.
- No reemplazar el verde oliva por cyan, azul neón o colores juveniles.
- No abusar de gradientes brillantes.
- No mezclar demasiados acentos en una misma pantalla.
- No usar saturación alta que rompa la sobriedad de la marca.

## Ejemplos de aplicación

### Correcto

- hero oscuro en `#111411`;
- cards en `#242924`;
- títulos en `#F3F4EF`;
- textos secundarios en `#A7AEA5`;
- botones CTA en `#5E6F5B`;
- hover en `#8FA28A`.

### Incorrecto

- fondo negro con botones cyan brillante;
- textos en gris muy bajo contraste;
- varios colores de acento compitiendo;
- gradientes futuristas exagerados.
