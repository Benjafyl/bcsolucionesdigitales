# Contexto de Marca para IA

## Resumen rápido

`B&C Soluciones` es la marca principal.

El descriptor oficial es:

`Soluciones Digitales`

La estructura visual recomendada es:

```text
B&C Soluciones
Soluciones Digitales
```

Visualmente debe priorizarse `B&C Soluciones` como nombre principal y `Soluciones Digitales` como descriptor secundario.

## Qué hace la marca

`B&C Soluciones` crea soluciones digitales prácticas para empresas reales.

Su foco está en:

- automatización de atención;
- chatbots para empresas;
- integraciones con WhatsApp Business;
- sitios web profesionales;
- sistemas internos simples;
- soluciones digitales personalizadas.

## Qué debe transmitir

La marca debe sentirse:

- seria;
- moderna;
- técnica;
- clara;
- profesional;
- cercana;
- confiable.

## Qué debe evitar

La marca no debe verse como:

- proyecto improvisado;
- marca infantil;
- startup futurista exagerada;
- estética gamer;
- marca con colores neón;
- empresa que vende humo.

## Posicionamiento

`B&C Soluciones se posiciona como una empresa joven y profesional que crea soluciones digitales prácticas para empresas reales.`

## Mensaje principal

`Soluciones digitales para empresas reales.`

## Mensaje secundario

`Tecnología práctica para empresas reales.`

## Paleta oficial

| Nombre | Hex | Uso |
| --- | --- | --- |
| Verde oliva profundo | `#5E6F5B` | color principal, botones, isotipo, acentos |
| Negro carbón | `#111411` | fondo principal |
| Gris grafito | `#242924` | cards, fondos secundarios |
| Blanco hueso | `#F3F4EF` | texto principal, fondos claros |
| Gris piedra | `#A7AEA5` | texto secundario, detalles |
| Verde salvia | `#8FA28A` | hover, detalles, acento suave |

## Tipografías

- `Sora` para títulos, logo y titulares.
- `Inter` para textos, interfaz, botones y formularios.

Alternativas aceptables:

- `Manrope`
- `Plus Jakarta Sans`
- `Montserrat`, con moderación
- `General Sans`

## Identidad visual

La estética debe ser:

- minimalista;
- sobria;
- moderna;
- profesional;
- oscura;
- clara en jerarquía;
- con detalles tecnológicos sutiles.

Evitar:

- robots;
- cerebros de IA;
- circuitos literales;
- neón;
- efectos 3D;
- gradientes agresivos;
- exceso de redondez;
- apariencia juvenil informal.

## Logo

El logo debe basarse en un monograma `BC`:

- simple;
- geométrico;
- legible;
- profesional;
- memorable;
- sobrio.

Versiones necesarias:

- horizontal;
- vertical;
- isotipo;
- clara;
- oscura;
- monocromática.

## Voz y tono

Hablar de `tú` al cliente.

El tono debe ser:

- profesional;
- claro;
- cercano;
- directo;
- confiable;
- técnico cuando haga falta, pero fácil de entender.

## Frases de referencia

- Creamos soluciones digitales pensadas para negocios reales.
- Automatizamos procesos para que tu empresa trabaje con más orden y menos tareas repetitivas.
- Diseñamos sitios web, sistemas y automatizaciones adaptadas a las necesidades de tu negocio.
- Te ayudamos a responder mejor, ordenar procesos y ahorrar tiempo con tecnología práctica.

## Reglas para futuras IAs

Si una IA genera componentes, textos o diseños para esta marca, debe:

1. usar el nombre oficial correcto;
2. respetar la paleta oficial;
3. respetar la jerarquía `B&C Soluciones` + `Soluciones Digitales`;
4. priorizar sobriedad y claridad;
5. evitar clichés visuales de IA;
6. hablar con tono profesional y cercano;
7. diseñar para negocios reales, no para una estética futurista de moda.
