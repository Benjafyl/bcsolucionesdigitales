# Guía de Logo

## Concepto general

El logo oficial de `B&C Soluciones` debe construirse a partir de un monograma elegante basado en las letras `B` y `C`.

El símbolo debe comunicar:

- equilibrio;
- precisión;
- confianza;
- estabilidad;
- claridad;
- profesionalismo;
- competencia técnica.

## Estructura principal

La versión principal del sistema de logo debe ser:

```text
[Monograma BC] B&C Soluciones
               Soluciones Digitales
```

## Versiones necesarias

El sistema de marca debe contemplar al menos estas versiones:

- logo principal horizontal;
- logo vertical;
- isotipo solo;
- versión clara para fondos oscuros;
- versión oscura para fondos claros;
- versión monocromática.

## Dónde debe funcionar

El logo debe poder usarse correctamente en:

- sitio web;
- WhatsApp Business;
- Instagram;
- LinkedIn;
- TikTok;
- propuestas comerciales;
- tarjetas;
- documentos;
- stickers;
- favicon.

## Características que debe cumplir el monograma

El monograma debe ser:

- simple;
- legible;
- geométrico;
- profesional;
- memorable;
- sobrio;
- adaptable a tamaños pequeños.

## Dirección recomendada

La dirección recomendada del logo es un monograma `BC` minimalista con un detalle sutil de sistema, conexión o automatización.

Ese detalle debe sentirse implícito, no literal.

Puede resolverse mediante:

- geometría;
- continuidad de trazos;
- unión elegante entre formas;
- espacio negativo;
- equilibrio entre curvas y estructura.

## Usos correctos

- usar la versión horizontal en header y piezas institucionales;
- usar el isotipo en favicon, avatar, foto de perfil o usos compactos;
- usar la versión clara sobre fondo oscuro;
- usar la versión oscura sobre fondo claro;
- respetar proporciones, márgenes y legibilidad.

## Usos incorrectos

- cambiar colores oficiales del logo;
- deformar proporciones;
- aplicar efectos 3D;
- usar brillos exagerados;
- agregar gradientes agresivos;
- aplicar sombras decorativas innecesarias;
- mezclar el símbolo con iconos de WhatsApp o IA;
- reescribir el nombre oficial;
- reemplazar el descriptor oficial por otro texto.

## Elementos prohibidos en el logo

- robots;
- cerebros de IA;
- burbujas de chat obvias;
- íconos de WhatsApp;
- circuitos literales;
- colores neón;
- estética gamer;
- caricaturas;
- efectos 3D;
- degradados excesivos.

## Criterio para evaluar futuras versiones del logo

Antes de aprobar una nueva versión del logo, debe validarse:

1. ¿Se reconoce claramente como `B&C`?
2. ¿Se ve profesional y sobrio?
3. ¿Funciona en tamaños pequeños?
4. ¿Respeta la paleta oficial?
5. ¿Puede usarse tanto en web como en documentos?
6. ¿Se siente moderno sin ser exagerado?
7. ¿Evita clichés visuales de IA y automatización?

## Recomendación de uso práctico

- `Header web`: logo horizontal.
- `Footer`: logo horizontal o compacto.
- `Favicon`: isotipo.
- `Redes sociales`: isotipo.
- `Documentos y propuestas`: logo horizontal.
- `Aplicaciones muy pequeñas`: isotipo o versión monocromática.
