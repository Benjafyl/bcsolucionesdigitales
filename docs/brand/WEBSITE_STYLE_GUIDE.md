# Guía de Estilo Web

## Objetivo del sitio

El sitio web de `B&C Soluciones` debe servir como:

- carta de presentación comercial;
- respaldo de legitimidad para clientes y plataformas externas;
- base para captar contactos;
- puerta de entrada a futuras soluciones, automatizaciones y sistemas.

## Sensación general

El sitio debe verse:

- profesional;
- moderno;
- minimalista;
- serio;
- confiable;
- claro.

## Dirección visual principal

La versión principal del sitio puede usar:

- fondo oscuro;
- secciones bien espaciadas;
- textos claros;
- acentos en verde oliva;
- bloques sobrios;
- detalles tecnológicos sutiles.

## Reglas visuales

### Fondo

- Priorizar `Negro carbón` como base principal.
- Usar `Gris grafito` en bloques secundarios y cards.
- Mantener contraste adecuado con `Blanco hueso` y `Gris piedra`.

### Cards

- discretas;
- limpias;
- con contraste claro;
- sin exceso de decoración;
- alineadas con una lógica corporativa.

### Botones

- sobrios;
- claros;
- con suficiente jerarquía;
- usar `Verde oliva profundo` como CTA principal;
- usar `Verde salvia` en hover o estados secundarios cuando corresponda.

### Bordes y radios

- bordes suaves, no exagerados;
- evitar sensación infantil por exceso de redondez;
- priorizar sobriedad y precisión.

### Espaciado

- generoso;
- limpio;
- consistente;
- pensado para facilitar lectura y escaneo.

### Jerarquía

- títulos claros;
- subtítulos breves;
- descripciones concretas;
- CTAs visibles;
- bloques fáciles de entender.

## Elementos recomendados

- secciones limpias;
- cards discretas;
- íconos lineales;
- mockups de sistemas o interfaces;
- capturas reales cuando existan;
- botones sobrios;
- fondos oscuros con contraste adecuado;
- detalles tecnológicos sutiles;
- estructura clara y fácil de entender.

## Elementos a evitar

- diseño recargado;
- estética infantil;
- exceso de bordes redondeados;
- estética gamer;
- neón;
- visual excesivamente futurista;
- gradientes agresivos;
- demasiadas animaciones llamativas;
- componentes que parezcan juguetones o poco corporativos.

## Estructura recomendada del sitio inicial

- Inicio
- Qué hacemos
- Servicios
- Beneficios
- Casos de uso
- Sobre nosotros
- Contacto
- Botón a WhatsApp
- Política de privacidad
- Términos o aviso legal

## Criterio de aprobación de UI

Una pantalla, sección o componente debería aprobarse solo si:

1. se entiende rápido;
2. transmite profesionalismo;
3. se siente alineado al branding;
4. no parece improvisado;
5. podría mostrarse con confianza a un cliente o a una plataforma como Meta.
