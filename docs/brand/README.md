# Documentación de Marca

Esta carpeta reúne la documentación oficial de branding de `B&C Soluciones` para mantener coherencia visual, verbal y estratégica en el sitio web, redes sociales, propuestas comerciales y futuros desarrollos.

## Objetivo

Estos documentos existen para que cualquier desarrollador, diseñador, colaborador o IA pueda entender rápidamente:

- cuál es el nombre oficial de la marca;
- cómo debe verse;
- cómo debe hablar;
- qué debe transmitir;
- qué decisiones visuales y de contenido respetan la identidad oficial;
- qué errores deben evitarse.

## Alcance

Esta documentación cubre:

- identidad general de la marca;
- posicionamiento;
- personalidad;
- paleta oficial;
- tipografías;
- reglas de uso del logo;
- voz y tono;
- lineamientos de sitio web;
- reglas de contenido;
- contexto resumido para IAs.

## Regla principal

La marca debe sentirse:

- seria;
- moderna;
- técnica;
- clara;
- profesional;
- cercana;
- útil para negocios reales.

La marca no debe sentirse improvisada, infantil, exageradamente futurista ni marketinera.
